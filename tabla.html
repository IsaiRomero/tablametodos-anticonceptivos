METODOS ANTICONCEPTIVOS
Método Descripción Eficacia
Píldoras anticonceptivas Se toman diariamente para prevenir el embarazo. Alta eficacia si se usan correctamente.
Condón Una funda de látex que se coloca sobre el pene para evitar el contacto entre los espermatozoides y el óvulo. Eficacia moderada.
Dispositivo Intrauterino (DIU) Un pequeño dispositivo de plástico o metal que se inserta en el útero para prevenir la fertilización. Alta eficacia y duración prolongada.
